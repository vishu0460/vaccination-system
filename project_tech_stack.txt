--------------------------------------------------
SUMMARY:
This project is a full-stack vaccination management and booking platform with separate React/Vite frontend and Spring Boot backend applications. It uses Java 17, Spring Boot 3, Spring Security, Spring Data JPA, MySQL, Flyway, React 18, Bootstrap, Axios, Chart.js, Recharts, and Docker-based deployment infrastructure with Nginx reverse proxying.

Key implemented features include citizen registration and login, JWT-based authentication with refresh flow, role-based access control, email verification, password reset, two-factor verification, vaccination center and drive discovery, slot booking and rescheduling, certificate generation and verification with QR codes and PDF export, admin analytics dashboards, notifications, reviews, feedback/contact workflows, SEO support, PWA/service-worker support, audit logging, caching, and automated testing across backend and frontend layers.
